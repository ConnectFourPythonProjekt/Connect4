# Connect4

Connect four is a 7x6 board game with two players. The objective of the game is to be the first to form a horizontal, vertical, or diagonal line of four of one's own discs.

We (Nikol Stefanova and Veronika Dimitrova)had to implement the utility functions, sufficient to play against another human. Then to design and implement an agent that uses mini-max with alpha-beta pruning and a simple heuristic and last but not least to Design and implement an agent that uses monte carlo tree search. The main objective of the course was to teach basic concepts of 'software carpentry' as Test-driven development, documentation, issue tracking, and milestones for project planning, debugging, profiling, and optimization.
